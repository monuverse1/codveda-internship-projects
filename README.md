# Task 1: Web Scraping Project

This project was completed as part of the Codveda Data Science Internship.

## Objective
To collect quotes and author names from a website using web scraping techniques.

## Tools Used
- Python
- BeautifulSoup
- Requests
- Pandas

## Project Description
The script extracts quotes and their authors from the website:
http://quotes.toscrape.com/

The scraped data is then stored in a structured format (CSV file).

## Output
The output file `quotes_data.csv` contains two columns:
- Quote
- Author

## Files in this repository
- web_scraping_task1.ipynb → Python code for scraping
- quotes_data.csv → Scraped dataset
